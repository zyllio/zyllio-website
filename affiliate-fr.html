L'inscription au programme d'affiliation est gratuite.

Qui peut postuler ?

Quand suis-je payé ?

Comment le suivi de mes recommandations est-il assuré ?

Le suivi est entièrement automatisé. Vous avez deux options pour lier un client à votre compte :

Le lien affilié : C'est la méthode la plus simple. Le client clique sur votre lien personnalisé et un cookie enregistre votre identifiant. S'il s'abonne, il vous est automatiquement attribué.

Le code affilié : Vous pouvez également donner un code spécifique que le client saisit lors de son inscription.

Où trouver ces éléments ? Votre lien unique et votre code de suivi sont disponibles à tout moment dans votre tableau de bord partenaire au sein de votre compte Zyllio.
